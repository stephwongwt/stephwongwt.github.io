  • 2023.01 - Present
    Systems Engineer
    Government Technology Agency, Singapore
    After the conclusion of the previous project in GovTech, I transitioned into a Systems engineer, and began working on a project to build an asset management system in the Open Digital Platform team. In this project, my role was more focused on frontend and some backend.

    A teammate and I explored the designs of the application using pen & paper wireframes and ExcaliDraw, basing the design on an existing UI/UX framework created in Figma by other project teams.

    We used Mantine's components to get started quickly on UI/UX, and developed the rest of the stack with ReactJS, Next.js, NGINX, Express.js, and Redis for caching. As we were already familiar with type-safety, we chose to use TypeScript instead of JavaScript. And after the basics of the app has been set up, we also integrated with Keycloak for OAuth2 authentication and role based access.

    The application was to be deployed in an Azure managed Kubernetes cluster, so after the infra team has everything set up, I worked on dockerizing our applications and then deploying them to the cluster.

    The main feature of the application is to allow users to manage their assets remotely, so we implemented real-time updates between the frontend and backend, with the backend connected to other services via AMQP and Protobuf.

    Aside from the app, I also helped build new features for a Unity based digital twin used for managing assets, and embedded the app in Unity as part of a complete dashboard.

    During the deployment of the dashboard, I also worked on UAT planning, and managing the test cycles with the end users, as well as working with our VAPT vendor to resolve any vulnerabilities that arise, following OWASP and any other internal security requirements.
